This is a file that describes all the modes users and channels may have

USERS
i - Makes the user appear offline
o - User is an operator						May ONLY be set with OPER
r - User is not registered					May ONLY be set with NICK
a - Marked as away							MAY ONLY be set with AWAY

CHANNELS
o - Give or remove chanop
s - secret channel
i - invite only
b - bans
v - Give or remove chanvoice
m - require chanvoice to speak
k - edit key
